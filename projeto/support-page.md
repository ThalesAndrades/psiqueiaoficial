## 🚀 Começando

### Como criar uma conta?

**Para Pacientes:**
1. Abra o app PsiquèIA
2. Toque em "Criar conta"
3. Selecione "Paciente"
4. Preencha nome, email e senha
5. Confirme seu email (verifique caixa de entrada)
6. Complete o onboarding

**Para Psicólogos:**
1. Abra o app PsiquèIA
2. Toque em "Criar conta"
3. Selecione "Psicólogo"
4. Preencha nome, email, senha e CRP
5. Confirme seu email
6. Complete seu perfil profissional
7. Configure recebimentos via Stripe

### Esqueci minha senha

1. Na tela de login, toque "Esqueceu a senha?"
2. Digite seu email
3. Verifique sua caixa de entrada (pode levar até 5 minutos)
4. Clique no link recebido
5. Defina uma nova senha

## 👤 Para Pacientes

### Como agendar uma sessão?

1. Acesse a aba "Agendar"
2. Escolha um psicólogo da lista
3. Veja disponibilidade no calendário
4. Selecione data e horário
5. Confirme o agendamento
6. Realize o pagamento via cartão de crédito

### Como pagar uma sessão?

- Pagamentos são processados via Stripe (seguro e criptografado)
- Aceitamos: Visa, Mastercard, American Express, Elo
- Cartão de teste para demonstração: `4242 4242 4242 4242`

### Como acessar minha sessão?

1. Vá para "Agenda"
2. Encontre a sessão agendada
3. No horário marcado, toque "Entrar na Sessão"
4. Você será redirecionado para o Google Meet
5. Autorize câmera e microfone

### Como usar o diário emocional?

1. Acesse "Diário" no menu
2. Toque "+" para nova entrada
3. Selecione seu humor atual
4. Escreva sobre seu dia
5. (Opcional) Compartilhe com seu psicólogo
6. Receba insights de IA

### Como compartilhar meu diário com meu psicólogo?

1. Abra uma entrada do diário
2. Toque no ícone de compartilhamento
3. Selecione seu psicólogo
4. Confirme compartilhamento

## 🧑‍⚕️ Para Psicólogos

### Como completar meu perfil?

1. Acesse "Perfil"
2. Toque "Editar Perfil"
3. Preencha:
   - Foto profissional
   - CRP (obrigatório)
   - Especialidades
   - Biografia
   - Preço de sessão
   - Horários de disponibilidade

### Como configurar recebimentos?

1. Acesse "Financeiro"
2. Toque "Configurar Recebimentos"
3. Você será redirecionado para o Stripe
4. Complete o cadastro:
   - Dados pessoais
   - Dados bancários
   - Documentos (RG, CPF, comprovante de endereço)
5. Aguarde aprovação (1-2 dias úteis)

### Como gerenciar pacientes?

1. Acesse "Pacientes"
2. Veja lista de todos os pacientes
3. Toque em um paciente para ver:
   - Histórico de sessões
   - Diário compartilhado
   - Plano de tratamento
   - Notas de sessão

### Como adicionar notas de sessão?

1. Após a sessão, vá para "Agenda"
2. Encontre a sessão realizada
3. Toque em "Adicionar Notas"
4. Escreva suas observações
5. Salve

### Como acessar meu dashboard financeiro?

1. Acesse "Financeiro"
2. Veja:
   - Saldo disponível
   - Receitas do mês
   - Histórico de transações
   - Status de pagamentos
3. Toque "Acessar Stripe Dashboard" para detalhes completos

## 💳 Pagamentos e Financeiro

### Como funciona o pagamento?

**Para Pacientes:**
- Pague via cartão de crédito no app
- Processamento seguro via Stripe
- Receba confirmação por email
- Valor é cobrado imediatamente

**Para Psicólogos:**
- Receba automaticamente após a sessão
- 90% do valor vai para você
- 10% é taxa da plataforma
- Dinheiro cai na conta em 2-7 dias úteis

### Posso cancelar uma sessão?

**Sim, mas:**
- Cancelamento até 24h antes: reembolso total
- Cancelamento com menos de 24h: sem reembolso
- Psicólogo pode cancelar sem penalidade (paciente recebe reembolso)

**Como cancelar:**
1. Vá para "Agenda"
2. Encontre a sessão
3. Toque "Cancelar"
4. Confirme o cancelamento

### Como solicitar reembolso?

1. Entre em contato: contact@psiqueia.com
2. Informe o ID da sessão
3. Explique o motivo
4. Aguarde análise (1-3 dias úteis)
5. Reembolso em até 10 dias úteis

## 🔒 Privacidade e Segurança

### Meus dados estão seguros?

**Sim!**
- Todos os dados são criptografados
- Conformidade com LGPD
- Servidores seguros (Supabase)
- Não compartilhamos seus dados com terceiros

### Como deletar minha conta?

1. Acesse "Perfil"
2. Role até o final
3. Toque "Deletar Conta"
4. Confirme a ação
5. Digite sua senha
6. Todos os seus dados serão deletados em 30 dias

**⚠️ Atenção:** Esta ação é irreversível!

### Como exportar meus dados?

1. Entre em contato: privacy@psiqueia.com
2. Solicite exportação de dados
3. Receberá um arquivo JSON em até 15 dias úteis

## 📱 Problemas Técnicos

### App não abre / Tela branca

1. Force-close o app
2. Reabra o app
3. Se persistir, reinicie o dispositivo
4. Se ainda não funcionar, reinstale o app

### Não consigo fazer login

**Email ou senha incorretos:**
- Verifique se digitou corretamente
- Use "Esqueceu a senha?" se necessário

**Conta não confirmada:**
- Verifique email de confirmação
- Confira pasta de spam
- Solicite novo email: login → "Reenviar confirmação"

**Outro erro:**
- Tente em outra rede (WiFi/4G)
- Limpe cache do app
- Reinstale o app

### Notificações não chegam

**iOS:**
1. Configurações → PsiquèIA
2. Ative "Permitir Notificações"
3. Escolha tipo de alerta

**Android:**
1. Configurações → Apps → PsiquèIA
2. Notificações → Ativar

### Google Meet não abre

1. Verifique conexão com internet
2. Atualize Google Meet
3. Autorize câmera e microfone:
   - iOS: Ajustes → PsiquèIA → Ativar Câmera/Microfone
   - Android: Configurações → Apps → PsiquèIA → Permissões

### Pagamento falhou

**Verifique:**
- Saldo suficiente no cartão
- Dados do cartão corretos
- Cartão não está bloqueado
- Internet estável

**Ainda com problema:**
- Tente outro cartão
- Entre em contato com seu banco
- Email: contact@psiqueia.com

## ❓ Perguntas Frequentes (FAQ)

### O app é gratuito?

Sim, o download do app é gratuito. Você paga apenas pelas sessões de terapia agendadas.

### Quais são as formas de pagamento?

Cartão de crédito (Visa, Mastercard, American Express, Elo) via Stripe.

### Posso trocar de psicólogo?

Sim, você pode agendar sessões com diferentes psicólogos a qualquer momento.

### Os psicólogos são verificados?

Sim, todos os psicólogos passam por verificação de CRP antes de serem aprovados.

### Quanto tempo dura uma sessão?

Geralmente 50 minutos, mas pode variar conforme o psicólogo.

### Posso usar pelo computador?

Atualmente apenas app móvel (iOS e Android). Versão web em desenvolvimento.

### Há desconto para múltiplas sessões?

Entre em contato com seu psicólogo para pacotes personalizados.
